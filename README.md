## Tech Stack
Framework: Next.js 16+ (App Router)

Styling: Tailwind CSS (with custom glowing shadows and gradients)

Language: TypeScript

Components: Custom "GlassCard" components designed for this project.

## Getting Started
First, run the development server:

```Bash
pnpm install
pnpm dev
```

Open http://localhost:3000 with your browser to see the result.
